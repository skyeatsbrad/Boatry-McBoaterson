# Survivor

A Vampire Survivors-style game built with Python and Pygame.

## Features
- **3 Characters**: Knight, Mage, Rogue — each with unique weapons
- **Multiple Weapons**: Projectile, Lightning, Boomerang, AoE Explosion
- **Boss Fights**: Every 5 waves
- **Dash Ability**: Press SPACE to dodge
- **Persistent Progression**: Gold, upgrades, achievements, high scores
- **Particle Effects, Screen Shake, Sound Effects**
- **Auto-Update**: Checks GitHub for new versions on launch

## How to Play
- **WASD / Arrow Keys** — Move
- **SPACE** — Dash
- **Esc** — Pause
- **F11** — Fullscreen
- **F3** — FPS counter

Download the latest release from the [Releases](https://github.com/skyeatsbrad/Boaty-McBoaterson/releases) page.

## For Developers
Requires Python 3.10+ and Pygame:
```
pip install pygame
python game.py
```
